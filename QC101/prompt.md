### **Improved Prompt:**

**Objective:**  
Generate a comprehensive set of key notes from the provided video, ensuring no important information is omitted.

**Instructions:**

1.  **Complete Review:**  
    Confirm you have watched the **entire video** from start to finish before beginning analysis.

2.  **Multi-Source Analysis:**  
    Synthesize information from **all available sources**:
    *   **Visual Content:** Analyze all on-screen text, graphics, data visualizations, and slides.
    *   **Narration/Dialogue:** Use the full transcript to capture spoken explanations, emphasis, and context.
    *   **Structural Cues:** Note any chapter markers, section titles, or explicit transitions in the video.

3.  **Structural Breakdown:**  
    First, identify the video's **overall structure**. Determine:
    *   How many main sections or topics are covered?
    *   What is the logical flow or narrative progression?

4.  **Note-Generation Guidelines:**  
    Create key notes that are:
    *   **Comprehensive:** Cover every main section and subtopic identified.
    *   **Hierarchical:** Use headings and bullet points to reflect the video's structure (e.g., Main Section → Key Points → Supporting Details).
    *   **Concise & Clear:** Paraphrase for brevity while preserving meaning, technical terms, and key data.
    *   **Balanced:** Integrate insights from both the visual aids (slides) and the spoken commentary.

5.  **Final Verification:**  
    Before delivering the notes, perform a final check to ensure:
    *   All major sections have been addressed.
    *   Critical details from slides and transcript are incorporated.
    *   The summary stands as a complete, self-contained overview of the video's content.

**Output Format:**  
Please provide the notes in a well-organized markdown format with clear headings, subheadings, and bullet points.

---

### **Why This Prompt is More Effective:**

*   **Clear Objective:** Starts with a direct statement of the goal.
*   **Structured Steps:** Breaks down the complex task into logical, sequential instructions.
*   **Explicit Commands:** Uses strong action verbs (Synthesize, Identify, Create, Paraphrase, Ensure).
*   **Quality Checks:** Integrates verification at multiple stages (complete watch-through, final check).
*   **Defined Output:** Specifies the desired format, making the result more usable.
*   **Focus on Synthesis:** Emphasizes combining visual and auditory information, which is crucial for accurate summarization.

This version gives the AI (or a human) a precise, methodical framework to follow, greatly increasing the likelihood of a thorough and well-structured summary.
